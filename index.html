How it works

3ステップで始めるクセ改善

難しい設定は一切不要。アプリを開いてスタートするだけで、最先端のAIがあなたの習慣を見守ります。

Step 01

活動を選んで記録開始

勉強・PC作業・読書など、今やっていることを選んでスタート。iPhoneをスタンドに立てて、カメラを自分に向けるだけでOKです。

Step 02

AIがリアルタイムで監視

TrueDepth カメラが顔の微細な表情筋を解析。口が3秒以上開いた状態を検知すると、作業の邪魔にならないバイブで優しく通知します。

Step 03

セッション後に分析を確認

検知回数・無意識に開いていた時間・改善スコアが自動で集計されます。活動ごとの自分のクセの傾向をAIが客観的に分析します。

Features

習慣を変える、本格的な機能

単なるタイマーではありません。無意識の習慣を自覚し、改善へと導くために設計されたプレミアムな機能群です。

リアルタイム検知

Apple TrueDepth API × ARKit による高精度な顔認識技術。独自のEMAスムージングにより一時的なノイズを排除し、本当にクセが出た時だけ正確に捉えます。

スマートな通知システム

クセを検知した瞬間に触覚フィードバック（バイブレーション）で通知。カフェなど静かな場所でも使えるサイレントモードと、音付き通知を柔軟に切り替えられます。

高度なセッション分析

検知回数だけでなく、開口時間や平均気づき速度を美しいグラフで可視化。「読書中に発生しやすい」など、活動別の傾向をAIが自動でレポートします。

完全オフライン処理の安心設計

すべてのAI画像処理はあなたのiPhoneデバイス内で完結します。顔の映像データが外部サーバーに送信されることは物理的にあり得ません。プライバシーを最優先しています。

自動スナップショット記録

クセが検知された瞬間に、自動で顔写真をキャプチャ（記録のオン・オフ設定可能）。客観的に自分の状態を後から振り返ることで、改善への意識が飛躍的に高まります。

連続達成ストリーク

ゲーム感覚で毎日の記録を習慣化。連続日数を計測し、バッジでお祝いすることで、モチベーションを維持しながら長期的なクセとの決別をサポートします。

ポリシーとサポート

【最重要事項】すべての情報処理はあなたのiPhoneデバイス内で完結します。カメラ映像や顔認識用の深度データが外部のサーバーに送信されることは意図的にもシステム的にも存在しません。

1. TrueDepth API（顔認識データ）の利用について

Apple が提供する TrueDepth API（ARKit）を使用して、高精度な顔の表情データを取得します。

  • 利用目的：セッション実行中の口の開閉状態をリアルタイムで判別し、無意識のクセを検知した際にバイブレーション等のフィードバックを返すという、アプリのコア機能の提供目的のみに使用します。
  • データの取り扱い：取得した顔の深度データおよびカメラ映像ストリームは、その瞬間のリアルタイム判定計算にのみメモリ上で使用され、永続化（保存）や、インターネットを経由した外部サーバーへの送信、第三者への共有は一切行われません。

2. 記録データのローカル環境保存

アプリ内でユーザーが自発的に記録するデータ（セッションごとのクセ検出回数、時間、分析用ログ、および任意設定による顔のスナップショット写真）はすべてユーザー自身のiPhone端末本体のローカルストレージにのみ保存されます。ユーザーが本アプリをアンインストールした場合、これらのデータはOSの仕組みによりデバイスから完全に削除されます。

3. サードパーティツールの不使用

当アプリは、ユーザーの行動履歴や行動統計、クラッシュログ等を無断で収集するような、外部の解析ツール（Google Analytics, Firebase Analytics, 各種広告SDK 等）を一切組み込んでいません。

4. ポリシーの改定と通知

法令・ガイドラインの変更、または新機能の追加に伴い、必要に応じて本プライバシーポリシーを改定することがあります。重要な変更があった場合は、App Store のアップデート内容（リリースノート）やアプリ内のお知らせ等で明確に明示します。
